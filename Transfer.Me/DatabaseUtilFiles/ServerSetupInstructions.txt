Instructions for Installing MariaDB and getting the database up and running.

1) Go to : https://mariadb.org/download/?rel=10.5.12&prod=mariadb&os=windows&cpu=x86_64&pkg=msi&mirror=accretive
	* This should be the correct version (10.5.12) for windows and x86_64 CPU
	
2) Install the program and copy my.ini into the "C:\Program Files\MariaDB 10.5" directory
	
3) Launch Command Prompt as Administrator and run the following command: 	
	sc create MySQL binpath= "\"C:\Program Files\MariaDB 10.5\bin\mysqld.exe\" --defaults-file=\"C:\Program Files\MariaDB 10.5\my.ini\" MySQL" start= auto

4) Create the directory C:\mariadb\data\

5) copy files from DBStarterFiles directory into the new directory created.

6) create a the database called "transfer.me" using the queries in the TableCreateQueries.txt file, or use the .sql database dump file to import the database. You can use command line or a program like HeidiSQL or SQLYog to do this. I can include instructions if needed
